FatCatMap by sgammon

A simple visualizer for federal campaign contributions.

FatCatMap mashes up data from the Sunlight Foundation (http://www.sunlightfoundation.org) and the Center for Responsive Politics (http://www.opensecrets.org) to create a browsable, interactive graph of connections based on campaign contributions.

Dependencies

Built on Google's App Engine (http://code.google.com/appengine)

Install

If you choose to clone and push to your own GAE app, change the API keys in the Sunlight and OpenSecrets cron/queue files (in the services directory). You'll also need to replace the RemoteObject endpoints specified in Flex - the frontend viewer is available in another repository.
